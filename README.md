# Sales-Analysis-Dashboard
A Power BI Sales Analysis Dashboard that highlights sales performance, budget variance, customer segmentation, and monthly trends. Designed to give leadership a clear view of sales value, cost value, and customer behavior across key dimensions.
📘 Project Overview
The Sales Analysis Dashboard is a Power BI reporting solution designed to help organizations analyze sales performance, budget variance, customer segmentation, and monthly sales trends. It provides a complete, interactive view of sales KPIs and helps leadership understand where performance is strong and where improvements are needed.
